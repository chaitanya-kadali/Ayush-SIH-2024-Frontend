## Running Locally

1. Clone the repository:

```bash
git clone https://github.com/chaitanya-kadali/Ayush-SIH-2024-Frontend.git
```

2. Install dependencies
```bash
npm install
``` 
3. Start the application 
```bash
npm run dev
```
